# Investment Portfolio Optimization Tool

A hybrid investment portfolio optimizer that compares **Greedy**, **Dynamic Programming**, and **Branch & Bound** algorithms to help users maximize returns while minimizing risk.

## Features
- **Visual Portfolio Builder**: Add/remove stocks with real-time feedback.
- **Algorithm Comparison**: Compare three different algorithmic approaches based on your budget.
- **Risk-Return Visualization**: Interactive scatter plots using Chart.js.
- **Performance Metrics**: Comparative table showing execution time and total profit.
- **Modern UI**: Premium design with glassmorphism and smooth animations.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- Chart.js (Visualization)

## How to Run
1. Clone the repository.
2. Open `index.html` in your favorite browser.
3. (Optional) Run with a live server for the best experience.
