Greedy

Selects stocks with the highest return-to-price ratio first. Fast but may not always yield the optimal solution.

Dynamic Programming

Uses a 1D table to solve the 0/1 knapsack problem. Guaranteed optimal result for integer budgets.

Branch & Bound

Explores all combinations via DFS with pruning. Exact solution with better average-case performance than brute force.

Time: O(2ⁿ) worst
